# j-todolist

实现了核心的功能，一些其他的东西没有写，比如左滑右滑更新状态和删除，登录页没写，明暗切换没写（不会）
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
